# センサーテンプレート（sensor/）

## 概要

SPIKE Prime の各種センサーから値を読み取るテンプレート集です。
カラーセンサー、超音波距離センサー、フォース（力）センサーの基本的な
使い方と、測定値を使ったフィードバックの例を示します。

## テンプレート一覧

| ファイル | 説明 | 主な用途 |
|---|---|---|
| [01_color_sensor.py](01_color_sensor.py) | カラーセンサー（色判定・反射光・環境光・HSV・ライト制御） | 色の識別・ライントレースの下地 |
| [02_ultrasonic_sensor.py](02_ultrasonic_sensor.py) | 超音波距離センサー（距離計測・4 灯ライト制御） | 障害物検知・距離計測 |
| [03_force_sensor.py](03_force_sensor.py) | フォースセンサー（力・押し込み量・押下判定・接触判定） | ボタン入力・力の計測 |

## 各センサーの主なメソッド

### ColorSensor（カラーセンサー）
| メソッド | 説明 |
|---|---|
| `color()` | 最も近い色を判定（`Color.RED` など） |
| `hsv()` | 色相・彩度・明度の生の値 |
| `reflection()` | 反射光の強さ（0〜100%） |
| `ambient()` | 環境光の強さ（0〜100%） |
| `detectable_colors([...])` | 判定対象の色を限定して精度向上 |
| `lights.on((l, c, r))` | 前面 3 灯の明るさ設定 |

### UltrasonicSensor（超音波センサー）
| メソッド | 説明 |
|---|---|
| `distance()` | 距離 (mm)。測定不能時は 2000 |
| `presence()` | 他の超音波センサーの検出 |
| `lights.on((...))` | 前面 4 灯の明るさ設定 |

### ForceSensor（フォースセンサー）
| メソッド | 説明 |
|---|---|
| `force()` | 加えられた力 (N、最大約 10N) |
| `distance()` | 押し込み量 (mm、最大約 8mm) |
| `pressed(force=3)` | 指定の力以上で押されているか |
| `touched()` | わずかな接触の判定 |

## 接続ポートについて

テンプレートでは例として以下のポートを使用しています。実際の接続に合わせて変更してください。
- カラーセンサー: `Port.C`
- 超音波センサー: `Port.D`
- フォースセンサー: `Port.E`
